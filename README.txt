Что нового в версии v12:

1. Ельцовка-1 выделена отдельно как «Моя компания», а не как конкурент.
2. Появилось сравнение динамики: «Моя компания vs конкуренты».
3. Источник «Брокер» заменен на Avito.
4. База данных единая для всех пользователей одного запущенного бота: все ручные конкуренты и помещения сохраняются в общих файлах data/.
5. Теперь можно:
   - отдельно добавить название конкурента;
   - отдельно добавлять по нему ручные помещения;
   - удалять ошибочно добавленные ручные помещения;
   - удалять ручного конкурента вместе с его ручными помещениями.
6. Для ручных помещений введена ревизия:
   - через 14 дней помещение попадает в список на проверку;
   - после 28 дней без подтверждения помещение исключается из активной ручной базы и уходит в жизненный цикл как устаревшее.
7. Добавлены быстрые inline-кнопки под сообщениями.

Основные команды:
/start
/check
/report
/all
/allreport
/dynamic
/compare
/review
/priority
/archive
/changes
/add_competitor
/add_room
/delete_room
/delete_competitor
/cancel

Важно:
- чтобы у всех сотрудников были одинаковые данные, бот должен работать из одной общей развернутой копии;
- все данные хранятся в папке data/.
